基于docker的SRS部署运行步骤

发表于 2020-04-14 | 分类于 Github | 评论数： | 阅读次数：

SRS官网: http://ossrs.net/srs.release/releases/

SRS Github: https://github.com/ossrs/srs

1.安装docker

linux安装docker运行环境
windows安装docker for desktop程序(安装完毕可以通过cmd或powershell操作docker,命令与linux相同)

2.获取srs image

若服务器连通互联网环境:
    1)可以通过"docker search srs"命令检索srs镜像之后,通过执行:
        docker run -p 1935:1935 -p 1985:1985 -p 8080:8080 ossrs/srs
        进行容器生成
    2)执行执行以下命令获取srs官方镜像并生成容器:
        docker run -p 1935:1935 -p 1985:1985 -p 8080:8080 registry.cn-hangzhou.aliyuncs.com/ossrs/srs:3

若服务器无法连接互联网环境,则需要先获取srs的docker镜像:
    地址:http://ossrs.net/srs.release/releases/
    获取到对应镜像后加载,加载完毕后通过执行上面"1)"后的命令启动容器

3.srs不同功能的支持

srs可提供的功能由不同的conf文件提供,默认使用的文件为容器中usr/local/srs/conf下的srs.conf文件因此,如果需要启动某项功能,可以进入容器使用vi对srs.conf文件进行修改后重启容器.
conf文件的修改参考网址:
    https://github.com/ossrs/srs/wiki/v2_CN_Home
    具体为网页中Deployment Guides这一项中的内容参考.
